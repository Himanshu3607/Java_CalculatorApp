# Java Calculator App

A simple calculator application built using Java.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous calculations using a menu-driven program

## Technologies Used
- Java
- Git
- GitHub
- VS Code

## How to Run

1. Compile the program:
   javac src/Calculator.java

2. Run the program:
   java Calculator
